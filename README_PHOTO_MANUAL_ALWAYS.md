# Photo Manual Score Always Visible Fix

This version keeps OCR, but adds a manual score save panel that is always visible.

Why:
Some eFootball screenshots put the eFootball logo between the score numbers, so browser OCR may fail.

Admin → Photo Result Upload now shows:
1. Upload result screenshot
2. Manual score save

Use Manual score save when OCR says "No score detected":
- Choose the match
- Type home score
- Type away score
- Click Save Manual Score
